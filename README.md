# msr-lda-wordclouds
Topic Modeling using LDA, and visualization using wordclouds on the [Munich Security Report 2017](http://report.securityconference.de/). The text data extracted from the pdf is present in "./munichsr". The script generates word clouds for the top 50 terms in 14 topics from the extracted text using [scikitlearn LDA](http://scikit-learn.org/dev/modules/generated/sklearn.decomposition.LatentDirichletAllocation.html#sklearn.decomposition.LatentDirichletAllocation). I chose 14 because there were 14 articles including the Foreword in the report. The generated wordclouds for the topics are as follows:
